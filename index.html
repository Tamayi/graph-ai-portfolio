About Me

I am a Software Developer, Data Specialist and Cloud Solutions Architect with over two decades of experience delivering scalable solutions across healthcare, public health, and enterprise sectors — spanning 47 countries in my current role at the World Health Organization.

Course Modules

Module 1 · Week 1–2

Websites, Documents & Presentations

Learn how LLMs work and generate text responsibly for papers, grants, websites, and presentations.

  • How LLMs are trained; overview of popular providers
  • Generating Markdown, LaTeX and HTML outputs
  • Building research websites with ChatGPT Canvas & Replit
  • AI presentation tools for scientific communication
Module 2 · Week 2–3

AI-Automated Data Analysis

Master Julius, ChatGPT Code Interpreter and Windsurf for ethical automated data analysis and reporting.

  • ChatGPT Code Interpreter for quick data analysis
  • Automated dashboards with TheBricks
  • Batch file conversion and manipulation
  • Spreadsheet automations for recurring analyses
Module 3 · Week 3–4

GenAI for Research & Text Analysis

Literature search, text classification, sentiment analysis, and knowledge synthesis using AI.

  • Efficient literature reviews with Elicit & LiteRev
  • Text classification for document organisation
  • Knowledge maps from document collections
  • Automated summarisation workflows
Module 4 · Week 4–5

Custom GPTs and Chatbots

Build chatbots for research assistance using RAG systems, API integrations, and Botpress workflows.

  • LLM chat architecture, context windows & RAG
  • Building custom GPTs for specific use cases
  • Creating interactive chatbots with Botpress
  • Integrating knowledge bases with research materials
Module 5 · Week 5–6

AI for Task Automation

Design responsible AI agents for literature monitoring and orchestrate app-to-app workflows.

  • Introduction to AI automation and agent-based systems
  • Setting up email response automations
  • Building AI agents for complex automation flows
  • Browser control agents for web-based tasks
Module 6 · Week 6–7

GenAI for Communication & Outreach

AI-driven approaches for research dissemination: summaries, scientific figures, video abstracts.

  • AI-generated research summaries for different audiences
  • Scientific figures and diagrams with AI tools
  • AI-generated video abstracts and explanations
  • Content scheduling for research communication
Module 7 · Week 7–8

Capstone Project & Showcase

Create, refine, and present a final AI solution to peers and guests demonstrating responsible AI practice.

  • Project progress check & peer feedback sessions
  • Final project presentations
  • Public Q&A session
